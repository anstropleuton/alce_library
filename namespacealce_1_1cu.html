template<cu_compatible Container>
constexpr auto alce::cu::repeat ( const Container &  container,
long double  n 
)
inlineconstexpr

Repeat container n times.

Considering integer part of the number n as i, and fraction part as f. The container is repeated i.0 times, and then the container is added with subordinate container with round(0.f * container.size()) elements from the beginning.
