Detailed Description

File Utilities. Not what you are thinking.

Contains utilities regarding file handling, and functionality to handle binary files of SD format.

SD Format is a binary format which consists of chunks. Each chunk is a pair of Size and Data. The Size is the number of bytes of the raw data. The Data is the raw data, i.e., the memory map of an object.

Note
The SD Format does not specify the endianness of the data, neither does the code to handle this format. This implies that the binary file may not be portable for systems with different endianness.
