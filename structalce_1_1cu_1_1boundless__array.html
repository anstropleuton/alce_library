Detailed Description

template<typename element_type, std::size_t count>
requires (std::is_default_constructible_v<element_type>)
struct alce::cu::boundless_array< element_type, count >

A boundless array.

Index-access of this array always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
element_typeThe type of element.
countThe size of array.
