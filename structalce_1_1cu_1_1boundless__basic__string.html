Detailed Description

template<typename char_type, typename traits = std::char_traits<char_type>, typename alloc = std::allocator<char_type>>
requires (std::is_default_constructible_v<char_type>)
struct alce::cu::boundless_basic_string< char_type, traits, alloc >

A boundless basic string.

Index-access of this string always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
char_typeThe character type.
traitsThe character traits type.
allocThe allocator type, defaults to std::allocator<char_type> .
