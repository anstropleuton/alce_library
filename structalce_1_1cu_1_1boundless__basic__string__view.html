Detailed Description

template<typename char_type, typename traits = std::char_traits<char_type>>
requires (std::is_default_constructible_v<char_type>)
struct alce::cu::boundless_basic_string_view< char_type, traits >

A boundless basic string view.

Index-access of this string view always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.
