Detailed Description

template<typename element_type, std::size_t extents = std::dynamic_extent>
requires (std::is_default_constructible_v<element_type>)
struct alce::cu::boundless_span< element_type, extents >

A boundless span.

Index-access of this span always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
element_typeThe type of element.
extentsThe number of elements in the sequence, or std::dynamic_extent if dynamic (default).
