Detailed Description

template<typename element_type, typename alloc = std::allocator<element_type>>
requires (std::is_default_constructible_v<element_type>)
struct alce::cu::boundless_vector< element_type, alloc >

A boundless vector.

Index-access of this vector always returns a default constructed element when an invalid index is provided. Requires a default-constructible type.

Template Parameters
element_typeThe type of element.
allocAllocator type, defaults to std::allocator<T> .
